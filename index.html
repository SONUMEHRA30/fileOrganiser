HOW TO USE FSO ?

Generally in our computer system a large number of data files are present. So the storage and managment are the very neccesary things. In solution over that the FSO tool are very useful in this terms. By using FSO tool we can organized the all unsorted files in our system. Let us know what are the exact logic we are using for the Organizing method. In this file Organizing method the files are sort and move in the respective folder where the perticular file belongs. But this file sorting are done according to thier file extension. When we fire the neccesarry command for Organizing purpose the all files are sccaned by theire extension and after scanning the decision is made that is in which folder the file belongs, after making the decision the file are move in thire respective folder.

Step wise instruction for installing and handelling the FSO tool.

    STEP 1 (Installation)

  1. Firstly you need to install a Node js software for backend purpose.
    (🛑 Note: please download appropriate version of node.js according to your system.)
  2. After successfully installation of node.js, We need to install our FSO tool(File System Organizer tool).
    3. When installation of FSO tool is completed successfully, After that we need to open our FSO File containing folder in cmd(command prompt) of your computer sytem.
  3. After the opening of FSO folder in cmd then our actual command implementation will start.✅

    4. STEP 2 (Available Command)

  4. In this FSO tool we have three command. These three command is neccessary for the complition of organzing task in your system.
    Besides that all three command have a exact/proper format of firing in cmd.
  5. These three command are as following :
      I) Help Command (command format : node FO.js _help_)
      II) Organize Command (command format : node FO.js organize _directory name_)
      III) Tree Command (command format : node FO.js tree _directory name_)

    6. STEP 3 (Organizing)

  6. The first Help command is necessary to show the option available throughout throughout our program. For using Help command user need to fire a specific command i..e, node FO.js _help_
  7. Organize command done the job for user to organize the unsorted files in a sorted manner. Then implementation of organize function user needs to fire/write a specific command i.e, "node FO.js organize _directory path_".
  8. Similarly, For implementation of Tree command user needs to fire/write a specific command i.e, "node FO.js tree _directory path_". This tree function print the sorted files in a tree structure for the better understandings of storage of files. this will show the placement of files with respect to its folder.
